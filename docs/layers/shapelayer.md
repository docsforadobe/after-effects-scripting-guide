# ShapeLayer object

`app.project.item(index).layer(index)`

**Description**

The ShapeLayer object represents a shape layer within a composition. Create it using [LayerCollection.addShape()](layercollection.md#layercollection-addshape). It can be accessed in an itemâ€™s layer collection either by index number or by a name string.

> ShapeLayer is a subclass of [AVLayer](avlayer.md#avlayer), which is a subclass of [Layer](layer.md#layer). All methods and attributes of AVLayer and Layer are available when working with ShapeLayer.
